A Python-based tool that calculates total connected electrical load using appliance data stored in an ANSI SQL database. The system computes total wattage, required current, and recommends the correct MCB rating. Built using basic OOP concepts, SQLite (ANSI SQL), and a clean modular structure.

Features

Add appliances with wattage & quantity

Stores data using ANSI SQL (SQLite)

Calculates total load (W)

Computes required current (A)

Suggests appropriate MCB rating

Simple OOP-based design

Tech Stack

Python

SQLite (ANSI SQL)

Object-Oriented Programming

How to Run

Install Python

Run: 

python project.py


Follow the on-screen menu options
